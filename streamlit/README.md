# Aplicação Streamlit

Esta pasta contém a aplicação preditiva desenvolvida em Streamlit
para o Tech Challenge – Fase 4.

A aplicação permite que o usuário insira características do paciente
e obtenha como saída a previsão do nível de obesidade, auxiliando
a tomada de decisão da equipe médica.

A aplicação foi desenvolvida como prova de conceito (POC).
