# Machine Learning

Esta pasta contém os artefatos relacionados ao pipeline de Machine Learning
desenvolvido para o Tech Challenge – Fase 4.

O objetivo do modelo é prever o nível de obesidade a partir de dados
demográficos, comportamentais e de estilo de vida.

Os experimentos e o treinamento do modelo foram realizados em Python,
utilizando bibliotecas como Pandas e Scikit-learn.
