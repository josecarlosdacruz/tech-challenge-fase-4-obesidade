# Tech Challenge – Fase 4 – Obesidade

Projeto desenvolvido como parte da Pós-Tech em Data Analytics da FIAP.

---

## Objetivo

Desenvolver uma solução analítica e preditiva para apoiar a equipe médica
na compreensão e identificação de fatores associados à obesidade, utilizando
dados demográficos, comportamentais e de estilo de vida.

---

## Base de Dados

- `obesity.csv`  
Base de dados contendo informações demográficas, hábitos alimentares,
atividade física, estilo de vida e níveis de obesidade.

---

## Solução Desenvolvida

O projeto contempla os seguintes entregáveis:

- Pipeline de Machine Learning para predição do nível de obesidade
- Aplicação preditiva desenvolvida em Streamlit (prova de conceito)
- Dashboard analítico desenvolvido no Power BI com principais insights
  para apoio à tomada de decisão da equipe médica

## Dashboard Analítico

O dashboard analítico está organizado em três páginas:

1. **Visão Geral da Obesidade**  
   Indicadores-chave e distribuição da população por nível de obesidade.

2. **Fatores de Risco**  
   Análise de atividade física, alimentação, consumo de água e álcool.

3. **Estilo de Vida**  
   Avaliação de sedentarismo, tempo em telas e meio de transporte.

O painel foi desenvolvido com foco em clareza, interatividade e linguagem
de negócio, priorizando a interpretação clínica dos dados.
